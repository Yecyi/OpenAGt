### 袩芯谢褜蟹芯胁邪褌械谢褜褋泻懈械 懈薪褋褌褉褍屑械薪褌褘

袩谢邪谐懈薪褘 褌邪泻卸械 屑芯谐褍褌 写芯斜邪胁谢褟褌褜 胁 opencode 褋芯斜褋褌胁械薪薪褘械 懈薪褋褌褉褍屑械薪褌褘:

```ts title=".opencode/plugins/custom-tools.ts"
import { type Plugin, tool } from "@openagt/plugin"

export const CustomToolsPlugin: Plugin = async (ctx) => {
  return {
    tool: {
      mytool: tool({
        description: "This is a custom tool",
        args: {
          foo: tool.schema.string(),
        },
        async execute(args, context) {
          const { directory, worktree } = context
          return `Hello ${args.foo} from ${directory} (worktree: ${worktree})`
        },
      }),
    },
  }
}
```

袩芯屑芯褖薪懈泻 `tool` 褋芯蟹写邪械褌 褋芯斜褋褌胁械薪薪褘泄 懈薪褋褌褉褍屑械薪褌, 泻芯褌芯褉褘泄 屑芯卸械褌 胁褘蟹褘胁邪褌褜 opencode. 袨薪 锌褉懈薪懈屑邪械褌 褎褍薪泻褑懈褞 褋褏械屑褘 Zod 懈 胁芯蟹胁褉邪褖邪械褌 芯锌褉械写械谢械薪懈械 懈薪褋褌褉褍屑械薪褌邪:

- `description`: 效褌芯 写械谢邪械褌 懈薪褋褌褉褍屑械薪褌
- `args`: 褋褏械屑邪 Zod 写谢褟 邪褉谐褍屑械薪褌芯胁 懈薪褋褌褉褍屑械薪褌邪.
- `execute`: 褎褍薪泻褑懈褟, 泻芯褌芯褉邪褟 蟹邪锌褍褋泻邪械褌褋褟 锌褉懈 胁褘蟹芯胁械 懈薪褋褌褉褍屑械薪褌邪.

袙邪褕懈 锌芯谢褜蟹芯胁邪褌械谢褜褋泻懈械 懈薪褋褌褉褍屑械薪褌褘 斜褍写褍褌 写芯褋褌褍锌薪褘 写谢褟 芯褌泻褉褘褌懈褟 泻芯写邪 薪邪褉褟写褍 褋芯 胁褋褌褉芯械薪薪褘屑懈 懈薪褋褌褉褍屑械薪褌邪屑懈.

### 啸褍泻懈 褋卸邪褌懈褟

袧邪褋褌褉芯泄褌械 泻芯薪褌械泻褋褌, 胁泻谢褞褔邪械屑褘泄 锌褉懈 褋卸邪褌懈懈 褋械邪薪褋邪:

```ts title=".opencode/plugins/compaction.ts"
import type { Plugin } from "@openagt/plugin"

export const CompactionPlugin: Plugin = async (ctx) => {
  return {
    "experimental.session.compacting": async (input, output) => {
      // Inject additional context into the compaction prompt
      output.context.push(`
## Custom Context

Include any state that should persist across compaction:
- Current task status
- Important decisions made
- Files being actively worked on
`)
    },
  }
}
```

啸褍泻 `experimental.session.compacting` 褋褉邪斜邪褌褘胁邪械褌 写芯 褌芯谐芯, 泻邪泻 LLM 褋谐械薪械褉懈褉褍械褌 褋胁芯写泻褍 写谢褟 锌褉芯写芯谢卸械薪懈褟. 袠褋锌芯谢褜蟹褍泄褌械 械谐芯 写谢褟 胁薪械写褉械薪懈褟 泻芯薪褌械泻褋褌邪, 褋锌械褑懈褎懈褔薪芯谐芯 写谢褟 写芯屑械薪邪, 泻芯褌芯褉褘泄 斜褍写械褌 锌褉芯锌褍褖械薪 锌褉懈 蟹邪锌褉芯褋械 薪邪 褋卸邪褌懈械 锌芯 褍屑芯谢褔邪薪懈褞.

袙褘 褌邪泻卸械 屑芯卸械褌械 锌芯谢薪芯褋褌褜褞 蟹邪屑械薪懈褌褜 蟹邪锌褉芯褋 薪邪 褍锌谢芯褌薪械薪懈械, 褍褋褌邪薪芯胁懈胁 `output.prompt`:

```ts title=".opencode/plugins/custom-compaction.ts"
import type { Plugin } from "@openagt/plugin"

export const CustomCompactionPlugin: Plugin = async (ctx) => {
  return {
    "experimental.session.compacting": async (input, output) => {
      // Replace the entire compaction prompt
      output.prompt = `
You are generating a continuation prompt for a multi-agent swarm session.

Summarize:
1. The current task and its status
2. Which files are being modified and by whom
3. Any blockers or dependencies between agents
4. The next steps to complete the work

Format as a structured prompt that a new agent can use to resume work.
`
    },
  }
}
```

袝褋谢懈 褍褋褌邪薪芯胁谢械薪 `output.prompt`, 芯薪 锌芯谢薪芯褋褌褜褞 蟹邪屑械薪褟械褌 锌褉懈谐谢邪褕械薪懈械 薪邪 褋卸邪褌懈械 锌芯 褍屑芯谢褔邪薪懈褞. 袦邪褋褋懈胁 `output.context` 胁 褝褌芯屑 褋谢褍褔邪械 懈谐薪芯褉懈褉褍械褌褋褟.
