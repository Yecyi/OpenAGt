## 袣芯薪褎懈谐褍褉邪褑懈褟

袙褘 屑芯卸械褌械 锌械褉械写邪褌褜 芯斜褗械泻褌 泻芯薪褎懈谐褍褉邪褑懈懈 写谢褟 薪邪褋褌褉芯泄泻懈 锌芯胁械写械薪懈褟. 协泻蟹械屑锌谢褟褉 锌芯-锌褉械卸薪械屑褍 锌芯谢褍褔邪械褌 胁邪褕 `opencode.json`, 薪芯 胁褘 屑芯卸械褌械 锌械褉械芯锌褉械写械谢懈褌褜 懈谢懈 写芯斜邪胁懈褌褜 胁褋褌褉芯械薪薪褍褞 泻芯薪褎懈谐褍褉邪褑懈褞:

```javascript
import { createOpencode } from "@openagt/sdk"

const opencode = await createOpencode({
  hostname: "127.0.0.1",
  port: 4096,
  config: {
    model: "anthropic/claude-3-5-sonnet-20241022",
  },
})

console.log(`Server running at ${opencode.server.url}`)

opencode.server.close()
```

## 小褌褉褍泻褌褍褉懈褉芯胁邪薪薪褘泄 胁褘胁芯写

袙褘 屑芯卸械褌械 蟹邪锌褉芯褋懈褌褜 褋褌褉褍泻褌褍褉懈褉芯胁邪薪薪褘泄 胁褘胁芯写 JSON 芯褌 屑芯写械谢懈, 褍泻邪蟹邪胁 `format` 褋芯 褋褏械屑芯泄 JSON. 袦芯写械谢褜 斜褍写械褌 懈褋锌芯谢褜蟹芯胁邪褌褜 懈薪褋褌褉褍屑械薪褌 `StructuredOutput` 写谢褟 胁芯蟹胁褉邪褌邪 锌褉芯胁械褉械薪薪芯谐芯 JSON, 褋芯芯褌胁械褌褋褌胁褍褞褖械谐芯 胁邪褕械泄 褋褏械屑械.

### 袨褋薪芯胁薪芯械 懈褋锌芯谢褜蟹芯胁邪薪懈械

```typescript
const result = await client.session.prompt({
  path: { id: sessionId },
  body: {
    parts: [{ type: "text", text: "Research Anthropic and provide company info" }],
    format: {
      type: "json_schema",
      schema: {
        type: "object",
        properties: {
          company: { type: "string", description: "Company name" },
          founded: { type: "number", description: "Year founded" },
          products: {
            type: "array",
            items: { type: "string" },
            description: "Main products",
          },
        },
        required: ["company", "founded"],
      },
    },
  },
})

// Access the structured output
console.log(result.data.info.structured_output)
// { company: "Anthropic", founded: 2021, products: ["Claude", "Claude API"] }
```

### 孝懈锌褘 褎芯褉屑邪褌芯胁 胁褘胁芯写邪

| `json_schema` | 袙芯蟹胁褉邪褖邪械褌 锌褉芯胁械褉械薪薪褘泄 JSON, 褋芯芯褌胁械褌褋褌胁褍褞褖懈泄 锌褉械写芯褋褌邪胁谢械薪薪芯泄 褋褏械屑械        |

### 肖芯褉屑邪褌 褋褏械屑褘 JSON

袩褉懈 懈褋锌芯谢褜蟹芯胁邪薪懈懈 `type: 'json_schema'`, 褍泻邪卸懈褌械:

| 袩芯谢械         | 孝懈锌             | 袨锌懈褋邪薪懈械                                                               |
| ------------ | --------------- | ---------------------------------------------------------------------- |
| `type`       | `'json_schema'` | 袨斜褟蟹邪褌械谢褜薪芯. 校泻邪蟹褘胁邪械褌 褉械卸懈屑 褋褏械屑褘 JSON                                |
| `schema`     | `object`        | 袨斜褟蟹邪褌械谢褜薪芯. 袨斜褗械泻褌 JSON Schema, 芯锌褉械写械谢褟褞褖懈泄 褋褌褉褍泻褌褍褉褍 胁褘胁芯写邪         |
| `retryCount` | `number`        | 袧械芯斜褟蟹邪褌械谢褜薪芯. 袣芯谢懈褔械褋褌胁芯 锌芯胁褌芯褉薪褘褏 锌芯锌褘褌芯泻 锌褉芯胁械褉泻懈 (锌芯 褍屑芯谢褔邪薪懈褞: 2) |

### 袨斜褉邪斜芯褌泻邪 芯褕懈斜芯泻

袝褋谢懈 屑芯写械谢褜 薪械 屑芯卸械褌 胁褘写邪褌褜 写械泄褋褌胁懈褌械谢褜薪褘泄 褋褌褉褍泻褌褍褉懈褉芯胁邪薪薪褘泄 胁褘胁芯写 锌芯褋谢械 胁褋械褏 锌芯胁褌芯褉薪褘褏 锌芯锌褘褌芯泻, 芯褌胁械褌 斜褍写械褌 胁泻谢褞褔邪褌褜 `StructuredOutputError`:

```typescript
if (result.data.info.error?.name === "StructuredOutputError") {
  console.error("Failed to produce structured output:", result.data.info.error.message)
  console.error("Attempts:", result.data.info.error.retries)

```

### 袥褍褔褕懈械 锌褉邪泻褌懈泻懈

1. **袩褉械写芯褋褌邪胁谢褟泄褌械 褔械褌泻懈械 芯锌懈褋邪薪懈褟** 胁 褋胁芯泄褋褌胁邪褏 胁邪褕械泄 褋褏械屑褘, 褔褌芯斜褘 锌芯屑芯褔褜 屑芯写械谢懈 锌芯薪褟褌褜, 泻邪泻懈械 写邪薪薪褘械 懈蟹胁谢械泻邪褌褜
2. **袠褋锌芯谢褜蟹褍泄褌械 `required`**, 褔褌芯斜褘 褍泻邪蟹邪褌褜, 泻邪泻懈械 锌芯谢褟 写芯谢卸薪褘 锌褉懈褋褍褌褋褌胁芯胁邪褌褜
3. **袛械谢邪泄褌械 褋褏械屑褘 褋褎芯泻褍褋懈褉芯胁邪薪薪褘屑懈** 鈥?褋谢芯卸薪褘械 胁谢芯卸械薪薪褘械 褋褏械屑褘 屑芯谐褍褌 斜褘褌褜 褌褉褍写薪械械 写谢褟 锌褉邪胁懈谢褜薪芯谐芯 蟹邪锌芯谢薪械薪懈褟 屑芯写械谢褜褞
4. **校褋褌邪薪邪胁谢懈胁邪泄褌械 褋芯芯褌胁械褌褋褌胁褍褞褖懈泄 `retryCount`** 鈥?褍胁械谢懈褔懈胁邪泄褌械 写谢褟 褋谢芯卸薪褘褏 褋褏械屑, 褍屑械薪褜褕邪泄褌械 写谢褟 锌褉芯褋褌褘褏
