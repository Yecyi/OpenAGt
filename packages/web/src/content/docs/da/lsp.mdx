## Konfiguration

Du kan tilpasse LSP-servere gennem sektionen `lsp` i din opencode-konfiguration.

```json title="opencode.json"
{
  "$schema": "https://github.com/Yecyi/OpenAGt/raw/dev/packages/web/dist/config.json",
  "lsp": {}
}
```

Hver LSP- server understﾃｸtter fﾃｸlgende:

| Egenskab         | Type     | Beskrivelse                                               |
| ---------------- | -------- | --------------------------------------------------------- |
| `disabled`       | boolean  | Indstil dette til `true` for at deaktivere LSP-serveren   |
| `command`        | string[] | Kommandoen til at starte LSP-serveren                     |
| `extensions`     | string[] | Filtypenavne, som denne LSP-server skal hﾃ･ndtere          |
| `env`            | object   | Miljﾃｸvariabler, der skal indstilles, nﾃ･r serveren starter |
| `initialization` | object   | Initialiseringsmuligheder for at sende til LSP-serveren   |

Lad os se pﾃ･ nogle eksempler.

---

### Miljﾃｸvariabler

Brug egenskaben `env` til at indstille miljﾃｸvariabler, nﾃ･r du starter LSP-serveren:

```json title="opencode.json" {5-7}
{
  "$schema": "https://github.com/Yecyi/OpenAGt/raw/dev/packages/web/dist/config.json",
  "lsp": {
    "rust": {
      "env": {
        "RUST_LOG": "debug"
      }
    }
  }
}
```

---

### Initialiseringsmuligheder

Brug egenskaben `initialization` til at videregive initialiseringsindstillinger til LSP-serveren. Disse er serverspecifikke indstillinger sendt under LSP `initialize` anmodningen:

```json title="opencode.json" {5-9}
{
  "$schema": "https://github.com/Yecyi/OpenAGt/raw/dev/packages/web/dist/config.json",
  "lsp": {
    "typescript": {
      "initialization": {
        "preferences": {
          "importModuleSpecifierPreference": "relative"
        }
      }
    }
  }
}
```

:::note
Initialiseringsmuligheder forskellige afhﾃｦngige af LSP-serveren. Se din LSP-servers dokumentation for tilgﾃｦngelige muligheder.
:::

---

### Deaktivering af LSP-servere

For at deaktivere **alle** LSP-servere globalt, skal du indstille `lsp` til `false`:

```json title="opencode.json" {3}
{
  "$schema": "https://github.com/Yecyi/OpenAGt/raw/dev/packages/web/dist/config.json",
  "lsp": false
}
```

For at deaktivere en **specifik** LSP-server skal du indstille `disabled` til `true`:

```json title="opencode.json" {5}
{
  "$schema": "https://github.com/Yecyi/OpenAGt/raw/dev/packages/web/dist/config.json",
  "lsp": {
    "typescript": {
      "disabled": true
    }
  }
}
```

---

### Brugerdefinerede LSP-servere

Du kan tilfﾃｸje brugerdefinerede LSP-servere ved at angive kommandoen og filtypenavne:

```json title="opencode.json" {4-7}

  "lsp": {
    "custom-lsp": {
      "command": ["custom-lsp-server", "--stdio"],
      "extensions": [".custom"]

  }
```
