## Indstil en standard

For at indstille en af 窶銀掬isse som standardmodellen kan du indstille `model`-nﾃｸglen i din
OpenAGt konfig.

```json title="opencode.json" {3}

  "model": "lmstudio/google/gemma-3n-e4b"
```

## Konfiguration af modeller

Du kan konfigurere en models muligheder globalt gennem konfigurationen.

```jsonc title="opencode.jsonc" {7-12,19-24}
{
  "$schema": "https://github.com/Yecyi/OpenAGt/raw/dev/packages/web/dist/config.json",
  "provider": {
    "openai": {
      "models": {
        "gpt-5": {
          "options": {
            "reasoningEffort": "high",
            "textVerbosity": "low",
            "reasoningSummary": "auto",
            "include": ["reasoning.encrypted_content"],
          },
        },
      },
    },
    "anthropic": {
      "models": {
        "claude-sonnet-4-5-20250929": {
          "options": {
            "thinking": {
              "type": "enabled",
              "budgetTokens": 16000,
            },
          },
        },
      },
    },
  },
}
```

Her konfigurerer vi globale indstillinger for to indbyggede modeller: `gpt-5` nﾃ･r de tilgﾃ･s via `openai` udbyderen og `claude-sonnet-4-20250514` nﾃ･r de tilgﾃ･s via `anthropic` udbyderen.
Den indbyggede udbyder og modelnavne kan findes pﾃ･ [Models.dev](https://models.dev).

Du kan ogsﾃ･ konfigurere disse indstillinger for alle agenter, du bruger. Agentkonfigurationen tilsidesﾃｦtter alle globale muligheder her. [Learn more](/docs/agents/#additional).

Du kan ogsﾃ･ definere brugerdefinerede varianter, der udvider de indbyggede. Varianter giver dig mulighed for at konfigurere forskellige indstillinger for den samme model uden at oprette duplikerede poster:

```jsonc title="opencode.jsonc" {6-21}
{
  "$schema": "https://github.com/Yecyi/OpenAGt/raw/dev/packages/web/dist/config.json",
  "provider": {
    "opencode": {
      "models": {
        "gpt-5": {
          "variants": {
            "high": {
              "reasoningEffort": "high",
              "textVerbosity": "low",
              "reasoningSummary": "auto",
            },
            "low": {
              "reasoningEffort": "low",
              "textVerbosity": "low",
              "reasoningSummary": "auto",
            },
          },
        },
      },
    },
  },
}
```
